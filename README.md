# 🌾 Smart Geo-Intelligent Agriculture Decision Support System

## 📌 Project Overview

The Smart Geo-Intelligent Agriculture Decision Support System is an AI-powered web application developed to assist farmers in making informed agricultural decisions. The system predicts the most suitable crop based on environmental conditions, estimates crop yield, analyzes market prices, and provides weather-based recommendations to improve productivity and profitability.

The application integrates Machine Learning, real-time weather information, soil analysis, and agricultural datasets into a single platform.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Smart-Geo-Intelligent-Agriculture-Decision-Support-System.git
```

Navigate to the project

```bash
cd Smart-Geo-Intelligent-Agriculture-Decision-Support-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
